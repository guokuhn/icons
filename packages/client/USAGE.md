# 使用指南

## 快速开始

### 1. 启动 API 服务器

```bash
cd packages/api
npm run dev
```

API 服务器将在 `http://localhost:3000` 运行。

### 2. 启动客户端应用

```bash
cd packages/client
npm run dev
```

客户端应用将在 `http://localhost:5173` 运行。

### 3. 访问应用

在浏览器中打开 `http://localhost:5173`，你将看到：

- ✅ API 连接状态（绿色表示已连接）
- 📊 图标总数统计
- 🎨 所有可用图标的网格展示
- ➕ 上传图标按钮

## 上传图标

### 步骤 1: 打开上传表单

点击右上角的 **"上传图标"** 按钮，上传表单会展开。

### 步骤 2: 填写图标信息

1. **图标名称**：输入图标的唯一名称
   - 示例：`my-custom-icon`
   - 规则：只能包含字母、数字、连字符和下划线
   - 长度：1-50 个字符

2. **选择 SVG 文件**：点击文件选择器，选择你的 SVG 文件
   - 支持格式：`.svg` 或 `image/svg+xml`
   - 最大大小：1MB
   - 文件选择后会显示文件名和大小

3. **API Key**（可选）：
   - 默认值：`dev-api-key-12345`
   - 生产环境需要使用正确的 API Key

### 步骤 3: 上传

点击 **"上传图标"** 按钮。

### 步骤 4: 查看结果

- ✅ **成功**：显示绿色成功消息，图标列表自动刷新，新图标出现在网格中
- ❌ **失败**：显示红色错误消息，说明失败原因

## 查看图标

### 图标网格

所有图标以网格形式展示，每个图标卡片包含：

- 📷 **图标预览**：48x48 像素的图标显示
- 🏷️ **图标名称**：图标的唯一标识符
- 💻 **代码引用**：`gd:icon-name` 格式
- 📐 **尺寸信息**：图标的宽度和高度

### 交互效果

- **悬停**：鼠标悬停时，卡片会高亮显示蓝色边框和阴影
- **点击**：可以点击卡片（未来可以添加详情查看功能）

### 刷新列表

点击图标列表右上角的 **"🔄 刷新"** 按钮，重新从服务器加载图标列表。

## 使用图标

### 在 React 组件中使用

```tsx
import { Icon } from '@iconify/react';

function MyComponent() {
  return (
    <div>
      {/* 基础使用 */}
      <Icon icon="gd:my-icon" />
      
      {/* 自定义大小 */}
      <Icon icon="gd:my-icon" width="32" height="32" />
      
      {/* 自定义颜色 */}
      <Icon icon="gd:my-icon" style={{ color: '#1976d2' }} />
      
      {/* 在按钮中使用 */}
      <button>
        <Icon icon="gd:my-icon" width="20" height="20" />
        <span>点击我</span>
      </button>
    </div>
  );
}
```

### 图标命名空间

所有图标都使用 `gd` 命名空间，引用格式为：

```
gd:icon-name
```

例如：
- `gd:home`
- `gd:logo`
- `gd:menu`
- `gd:search`
- `gd:my-custom-icon`

## 常见问题

### Q: 上传失败，提示 "401 Unauthorized"

**A:** API Key 不正确。请确保使用正确的 API Key。开发环境默认为 `dev-api-key-12345`。

### Q: 上传失败，提示 "Invalid icon name format"

**A:** 图标名称格式不正确。请确保：
- 只包含字母、数字、连字符（-）和下划线（_）
- 长度在 1-50 个字符之间
- 不包含空格或特殊字符

### Q: 上传失败，提示 "File size exceeds the maximum limit"

**A:** SVG 文件太大。请确保文件大小不超过 1MB。

### Q: 图标上传成功但看不到

**A:** 尝试以下方法：
1. 点击 "🔄 刷新" 按钮
2. 刷新浏览器页面
3. 检查 API 服务器是否正常运行

### Q: 无法连接到 API 服务器

**A:** 请确保：
1. API 服务器正在运行（`npm run dev` 在 `packages/api` 目录）
2. API 服务器运行在 `http://localhost:3000`
3. 没有防火墙阻止连接

### Q: 如何删除图标？

**A:** 当前版本不支持通过 UI 删除图标。你可以：
1. 使用 API 直接删除：`DELETE http://localhost:3000/api/icons/gd/{icon-name}`
2. 或者直接删除文件系统中的图标文件

## 技术支持

如果遇到问题，请检查：

1. **浏览器控制台**：查看是否有 JavaScript 错误
2. **网络面板**：查看 API 请求是否成功
3. **API 服务器日志**：查看服务器端错误信息
