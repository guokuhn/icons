# 测试点击复制功能

## 测试步骤

1. **启动 React 应用**
   ```bash
   cd packages/client
   npm run dev
   ```

2. **打开浏览器**
   访问: http://localhost:5173

3. **测试复制功能**
   - 点击任意图标卡片
   - 观察右上角是否显示 "✓ 已复制" 提示
   - 打开文本编辑器，按 Ctrl/Cmd + V 粘贴
   - 验证粘贴的内容是否为 "gd:图标名称"

4. **测试多次复制**
   - 点击第一个图标
   - 等待提示消失
   - 点击第二个图标
   - 验证每次都能正确复制

5. **测试详情不触发复制**
   - 点击 "查看 SVG" 展开详情
   - 验证不会触发复制功能

## 预期结果

✅ 点击图标卡片后：
- 图标名称（格式：gd:icon-name）被复制到剪贴板
- 右上角显示绿色 "✓ 已复制" 提示
- 2秒后提示自动消失

✅ 鼠标悬停时：
- 卡片边框变蓝色
- 显示阴影效果
- 卡片轻微上移
- 显示提示文本 "点击复制: gd:icon-name"

✅ 点击详情时：
- 不触发复制功能
- 正常展开/收起 SVG 代码

## 浏览器兼容性测试

测试以下浏览器：
- ✅ Chrome/Edge (现代 Clipboard API)
- ✅ Firefox (现代 Clipboard API)
- ✅ Safari (现代 Clipboard API)
